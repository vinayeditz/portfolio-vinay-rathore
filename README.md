# 💼 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This portfolio showcases my skills, projects, education, achievements, and contact information in a clean and professional layout.

The main objective of this project is to create an attractive online portfolio that represents my journey as a Computer Science student and aspiring Software Developer.

---

## 🚀 Features

- ✨ Modern and clean user interface
- 📱 Fully responsive design
- 🧑 Hero section with profile image
- 📖 About Me section
- 💻 Technical Skills section
- 🚀 Projects showcase
- 🎓 Education timeline
- 🏆 Achievements & Certifications
- 📬 Contact section
- 🎨 Smooth scrolling navigation
- 🌐 Cross-browser compatible

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Responsive Web Design
